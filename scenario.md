# Сценарий игры <!-- omit in toc -->

Игроку предоставляется классическая головоломка Судоку — сетка 9×9, разделённая на 9 блоков 3×3.
Цель игры — заполнить все пустые клетки цифрами от 1 до 9 так, чтобы в каждой строке, каждом столбце и каждом блоке 3×3 каждая цифра встречалась только один раз.

## Общая механика

- После запуска игры игрок может:
  - Выбрать клетку на поле
  - Выбрать цифру для вставки под сеткой или ввести с клавиатуры
  - Использовать «заметки» (маленькие цифры внутри клетки)
  - Использовать подсказки
- Цель — **заполнить всю сетку корректно как можно быстрее**.
- Время прохождения сохраняется для таблицы рекордов.
- При успешном завершении — появляется модальное окно с результатом по времени и кнопками:
  - «Новая игра»
  - «Повторить игру»
  - «Вернуться в главное меню»

- При не успешном завершении — появляется модальное окно с кнопками:
  - «Начать заново»
  - «Вернуться в главное меню»

---

## Дополнения

- Игрок теряет жизнь при каждой ошибке - всего 3 жизни, при исчерпании — проигрыш и возможность начать заново.
- Цифра запишется **красным** цветом при неправельном значении.
- Игру можно открыть **На весь экран**.
